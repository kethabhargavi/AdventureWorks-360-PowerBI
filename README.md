# 📌 Overview

**AdventureWorks 360** is a comprehensive Business Intelligence platform built using **Power BI** to provide executive-level insights into sales, customer behavior, product performance, profitability, and geographical trends.

The project demonstrates an end-to-end BI workflow including data transformation, data modeling, DAX calculations, interactive visualizations, and AI-powered analytics to support strategic business decisions.

---

# 🎯 Project Objectives

* Build an executive-level interactive BI dashboard.
* Analyze sales performance across multiple business dimensions.
* Identify profitable products and regions.
* Evaluate customer purchasing behavior.
* Discover key business drivers using AI visuals.
* Deliver actionable recommendations for decision-makers.
